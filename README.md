# MasterApp
爬取研招网硕士专业的详细信息并导出EXCE文件

# 开发背景
在研招网上查询各个院校的硕士专业的详细信息（考试范围，拟招人数）非常麻烦，换个院校都需要重新进入查看，浏览器标签页多而乱
